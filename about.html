ABOUT DecDec BINGO:

 

"DecDec Bingo" online BINGO cards were created as an excercise in coding. Version 0 was written in a few hours on the day of December Decadence, 2011. It was only tested in Safari on an iPad 2 and the Firefox web browser on a Windows computer. It worked (albeit with some known issues) and took the place of many physical BINGO cards that night.

The original "DecDec Bingo" webpage did not include the "BINGO" title row which caused some confusion for some players.

One known (to the creator) issue was that if the webpage were to be swapped out of memory, the BINGO card would be lost. Unfortunately, one person did suffer from this.

For DecDec 2012, a method to restore the state of the card after a reload was added. In 2011, the "New Card" button was tiny, easily pressed, and did not ask for confirmation when activated; this was fixed for 2012. Also, the "BINGO" title row was added.

DecDec 2013 saw a minor revamp to correct some squashed rows caused by adding the "BINGO" title row but failing to adjust the row heights in 2012.

DecDec 2014 added an alert when the state of the card was unable to be stored (generally in "Private" or "Incognito" mode).

For DecDec 2015, the pseudo-random number generator in use was changed as 2014 saw a large number of simultaneous BINGOs and "blackout"s.

For DecDec 2016, this "about" page, a "help" page, an "(in)FAQ" page, and a "contact" page were added.
